# 体癖理論への招待

<InfoBox type="welcome" icon="🌟">
**体癖理論**は、野口晴哉によって確立された身体の個性を理解する体系です。
人間の身体には、それぞれ特有の動きや反応のパターンがあり、これを「体癖」と呼びます。
</InfoBox>

## なぜ体癖を学ぶのか？

体癖を理解することで、以下のような恩恵を受けることができます：

- **自分の特性を深く知る** - 身体的・精神的な個性の理解
- **他者との関係を改善する** - 相手の体癖を理解した適切なコミュニケーション
- **健康管理を最適化する** - 体癖に合った生活習慣や運動方法の発見
- **ストレス管理** - 自分の体癖特有の緊張パターンとその解消法を学ぶ

## 10種体癖の概要

体癖は大きく以下の5つの軸に分類されます：

<KeyPoint title="体癖の5つの軸">
1. **上下型（1種・2種）** - 頭部の使い方による分類
2. **左右型（3種・4種）** - 肩甲帯の動きによる分類  
3. **前後型（5種・6種）** - 骨盤の使い方による分類
4. **回転型（7種・8種）** - 体幹のねじれによる分類
5. **開閉型（9種・10種）** - 胸郭の動きによる分類
</KeyPoint>

## 体癖の組み合わせ

多くの人は単一の体癖ではなく、**主体癖**と**副体癖**の組み合わせを持っています：

<Callout type="info">
**主体癖**: 最も強く現れる体癖（例：3種）  
**副体癖**: 二番目に現れる体癖（例：7種）

この場合「3-7種」と表現され、3種の特徴を基調として7種の要素が加わります。
</Callout>
